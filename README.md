### 문제

- **Baekjoon** — 백준 온라인 저지
- **Programmers** — 프로그래머스
- **Leetcode** — NeetCode 150 진행 중 ([풀이 로그](leetcode/LOG.md))

### Leetcode 챌린지 규칙

- **"완료" 기준은 Accepted가 아니라 최적 시간복잡도 도달.** 통과만 했어도 최적 미달이면 실패로 본다.
- **결과 표기**: `✅ 컷 안` / `⏰ 컷 초과` / `❌ 실패(최적 미달·포기·최대 초과)`
- **시간 기준(컷 / 최대)**: Easy 15/30분 · Medium 30/60분 · Hard 45/90분
- `⏰` / `❌` 문제는 2회독 복습 리스트로 관리.
